## 坦克大战模块

        入口:
            JFrame 子类下的 main()

        角色包:
            我方坦克, 敌方坦克, 子弹对象, 爆破对象

        常量包:
            [基础设施] 地图布局, 图片资源, 敌我标记

        音乐包:
            音乐, 爆破音


## AWT & Swing

        - AWT: 抽象工具包, 早期用于编写 GUI, 基于本地方法 native, 速度快, 有平台兼容问题, 它把不同 OS 的差异化方法舍弃, 只留交集方法
        - Swing: 解决 AWT 存在的问题而新开发的 GUI 包, 改良, 扩展, 用 java 开发, 运行速度慢, 但跨 OS
